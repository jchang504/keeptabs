KeepTabs Options

Set hold key

The hold key is the key you hold while typing to activate your hotkeys. If it is used for native shortcuts in Chrome (e.g. Ctrl+f on Windows), creating an overlapping hotkey may or may not override the native shortcut. Operating system shortcuts, however, will not get overridden, so choose your hold key carefully to avoid side effects.

Customize hotkeys

Hit a Hotkey with your hold key to navigate to its Target. If Deduplicate is checked, KeepTabs will first check if the URL of any existing tabs starts with the Match Prefix, and if so, navigate to the first such existing tab (use the hotkey repeatedly to cycle through all matches). If not, or if there are no existing matching tabs, KeepTabs will open a new tab of the Target.

Syntax notes: If you omit the scheme (e.g. "http://", "https://",) it will match either of http or https. If you want to match "file://" or "chrome://" schemes, you must specify them. If you want to match to just a bare host with no specific path (e.g. "facebook.com"), your Match Prefix must still have a "/" after the host (e.g. "facebook.com/"). You will not be able to save if any of your match prefixes doesn't fit this requirement.
